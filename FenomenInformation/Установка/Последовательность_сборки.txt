4. FenomenModule
	FenomenModule.jar - главный файл для работы на удаленном модуле
	
	start.bat - командный файл для запуска программы 

	module.id - properties файл, который содержит код-идентификатор (текстовое представление ) 
данного модуля на сервере: select * from module where id_module=''
	
	settings.properties - настройки потоков, которые были получены от удаленного сервера 

	ComPortReaderWriter.jar - программа для обмена данными с COM-порта в безопастном режиме 
	( режим Proxy )

	com_port.settings - настройки для работы COM порта
(порты должны быть освобождены от блокировки локального Firewall-а) 

	/AlarmChecker - каталог для хранения полученных от сервера объектов по адресам в сети ModBus, 
которые определяют Alarm события

	/InformationChecker - каталог для хранения полученных от сервера объектов по адресам в сети ModBus, 
которые определяют Information события

	/HardPart/CP210x_VCP_Win2K_XP_S2K3.zip драйвер для COM порта, если используется виртуальное USB устройство
